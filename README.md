# LUNA25 Hybrid VGG + Performer Model

Dự án này triển khai một mô hình phân loại nodule phổi dựa trên dữ liệu LUNA25.  
Kiến trúc chính gồm:

- **VGG Backbone (2D CNN)**
- **Performer-style Efficient Attention Head**
- **Hybrid Fusion Layer**
- **5-Fold Cross Validation**
- **FastAPI Inference Server**
- **Docker Deployment**

Dự án được thiết kế để tham gia **LUNA25 Challenge** với mô hình phân loại nodule lành tính/ác tính dựa trên nodule block 3D được cắt sẵn.
